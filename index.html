La Guía de Pestañas de DevTools

Cuando abras la consola, verás varias pestañas. ¡No te asustes! Aquí tienes una guía rápida de las 4 más importantes.

Pestaña 1

Elements (Elementos)

Es el esqueleto interactivo de la página. Puedes ver y modificar el HTML y el CSS en tiempo real sin romper nada de forma permanente.

📌 Tarea clave: Cambiar el color de un botón o editar el texto de un párrafo inspeccionando la web.
VISTA PREVIA DE ELEMENTS HTML Vivo 
<div id="mascota" class="cute-pet anim-idle">
  <h1>¡Hola, soy Chroma!</h1>
  <p>Cambia mi texto desde el inspector.</p>
</div>
Pestaña 2

Tu canal directo de comunicación con JavaScript. Aquí puedes ver errores, imprimir logs y escribir código interactivo.

📌 Tarea clave: Ejecutar operaciones matemáticas, ver alertas o dar instrucciones.
CONSOLA INTERACTIVA Javascript

Tareas Básicas de Consola

Alimenta a Chroma usando los botones de la izquierda para disparar automáticamente estas tareas y observar el resultado en tu consola real.

console.log() Task #1

Muestra información general. Es el comando amigo que usarás un millón de veces para debugear.

console.log("🌸 ¡Chroma es genial!");
console.info() Task #2

Similar a log, pero semánticamente expresa que es un mensaje informativo.

console.info("ℹ️ Info: Refrescante.");
console.warn() Task #3

Muestra un triángulo amarillo. Alerta sobre posibles problemas sin romper la ejecución.

console.warn("⚠️ Cuidado con el ácido.");
console.error() Task #4

Genera un mensaje rojo llamativo y almacena el trazo de pila (stack trace) del error crítico.

console.error("❌ ¡Alerta crítica!");

💡 Usa los "Lanzadores Avanzados" debajo de la mascota para probar estos métodos.

console.table()

Transforma tus matrices y objetos en una hermosa tabla interactiva.

console.table([{ id: 1, req: "OK" }]);
console.assert()

Escribe un error SOLO si la condición dada es falsa.

console.assert(1 === 2, "🚨 Falló");
console.group()

Agrupa varios logs en un bloque desplegable para organizar la consola.

console.time()

Inicia un temporizador para medir el rendimiento de una operación.

Trucos de Devs Expertos

Lleva tus habilidades al siguiente nivel con estos secretos:

1. Añadir CSS a tus logs

Usa la bandera %c para que el navegador lea el segundo argumento como CSS.

console.log(
  "%c✨ Chroma ✨",
  "color: pink; font-weight: bold;"
);

2. Último valor con $_

Escribe $_ en tu consola y dará como resultado el valor de la última operación ejecutada.

3. Seleccionar nodo con $0

En la pestaña Elements inspecciona algo, luego ve a Console y escribe $0. Obtendrás ese elemento exacto.
